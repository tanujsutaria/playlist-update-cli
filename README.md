# Spotify Playlist Manager

A Python script to manage Spotify playlists through the command line.

## Features
- Add/refresh songs to a playlist
- Append songs to an existing playlist
- Extract playlist contents to a file

## Setup
1. Create a Spotify Developer account and register your application
2. Set environment variables:
   - SPOTIFY_CLIENT_ID
   - SPOTIFY_CLIENT_SECRET
   - SPOTIFY_REDIRECT_URI
